# 💸 Splity

A modern expense-sharing platform that helps friends, roommates, families, and travel groups split bills, track shared expenses, manage balances, and settle payments effortlessly.

## ✨ Features

- 👥 Create and manage groups
- 💰 Add shared expenses
- ⚖️ Split bills equally or with custom amounts
- 📊 Track balances in real time
- 💳 Record settlements
- 📅 View expense history
- 🔐 Secure authentication
- 📱 Responsive design
- ☁️ Supabase backend

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yashbajaj02/Splity.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
